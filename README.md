# python-E-commerce sales data-analytics-project

📝 Description

The Sales and Profit Analysis project focuses on analyzing a retail Superstore dataset to gain insights into sales performance, profitability, and customer segments. Using Python data analysis libraries, the project explores patterns in sales and profit across different categories, sub-categories, and time periods.

The goal of this project is to perform Exploratory Data Analysis (EDA) and create interactive visualizations that help understand business performance and support data-driven decision making.

🎯 Project Objectives

Analyze monthly sales and profit trends

Identify top-performing product categories and sub-categories

Understand customer segment contribution to revenue

Evaluate sales-to-profit ratio to measure profitability

🛠️ Technologies Used
Technology	Purpose
Python	Programming language
Pandas	Data cleaning and analysis
NumPy	Numerical operations
Plotly	Interactive data visualization
📂 Dataset Information

The dataset used in this project is the Superstore Sales dataset, which contains 9994 records and 21 columns related to customer orders and sales transactions.

Important Features

Order Date

Customer Segment

Category

Sub-Category

Sales

Profit

Quantity

Discount

Region

⚙️ Data Preprocessing

The following preprocessing steps were applied:

Converted Order Date and Ship Date to datetime format

Created new features:

Order Month

Order Year

Order Day of Week

Aggregated sales and profit using groupby operations

📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

1️⃣ Monthly Sales Analysis

A line chart was used to visualize sales trends across different months.

2️⃣ Sales by Category

A donut chart was created to analyze the contribution of each product category to total sales.

3️⃣ Sales by Sub-Category

A bar chart was used to identify which sub-categories generate the highest sales.

4️⃣ Monthly Profit Analysis

Profit trends were analyzed to understand seasonal variations.

5️⃣ Profit by Category

Compared profitability of different product categories.

6️⃣ Sales and Profit by Customer Segment

Analyzed the performance of Consumer, Corporate, and Home Office segments.

7️⃣ Sales-to-Profit Ratio

Calculated the ratio to evaluate how effectively sales convert into profit.

📈 Key Insights

The Consumer segment contributes the highest share of sales.

Some sub-categories show high sales but relatively low profit margins.

Profit trends vary across months, indicating possible seasonal demand patterns.

Discounts have a significant impact on overall profitability.
